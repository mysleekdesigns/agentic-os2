---
description: Run an Agent OS agent end-to-end and stream its transcript
---

Run the Agent OS CLI command `agent-os run $ARGUMENTS` from the repository root. The first argument is the agent id, the rest form the goal text. Stream the resulting transcript to the user. Forward any additional flags (such as `--json`, `--model`, `--no-mcp`, `--cwd`, `--provider`) verbatim.
